## Acceptance criteria

- Caméra live : bbox fluides via Frigate (`FRIGATE_LIVE=1`), pas de régression démo fichier (go2rtc).
- Alerte vitesse : `IsComplete` avec clip 6s + images ; `capture_source=frigate` si `EVIDENCE_BACKEND=frigate`.
- Plaque : slot `plate` = crop Frigate + OCR **PaddleOCR** (pas texte LPR Frigate).
- Comptage : `observation_mode` → pas de record Frigate.
- Rollback `FRIGATE_*=0` restaure la stack actuelle sans erreurs `frigate_sync`.
